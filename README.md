# Portafolio bootcamp analisis de datos C70 - 2025

Este portafolio contiene los proyectos desarrollados en TT.

## Contenido

### Sprint 1: Introduccion Python
 En este sprint revisamos los conceptos basicos de python tales como:
 - Variables.
 - Tipos de datos.
 - Listas.
 - Operaciones entre variables.

### Sprint 2: Continuación Python Basico

En este sprint vemos conceptos más avanzados de python como:
- Ciclos for/while
- Condicional if/else/elif.
- Listas anidadas.
